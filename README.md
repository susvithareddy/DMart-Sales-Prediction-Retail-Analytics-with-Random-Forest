# DMart Sales Prediction

Predicting item outlet sales for DMart stores using Machine Learning.

## Problem Statement
Predict the sales of each product at a particular store using historical data
including item attributes, outlet information, and sales figures.

## Tech Stack
Python | Pandas | NumPy | Scikit-Learn | Matplotlib | Seaborn | Random Forest

## Workflow
- Exploratory Data Analysis (EDA) on item & outlet features
- Missing value imputation (Item Weight, Outlet Size)
- Label Encoding & One-Hot Encoding for categorical variables
- Correlation analysis & feature selection
- Random Forest Regressor with train/test split
- Model evaluation using RMSE

## Dataset
Big Mart Sales Dataset (Train.csv & Test.csv)
